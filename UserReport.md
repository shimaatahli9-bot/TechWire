# تقرير مشروع مُراسِل

## تاريخ البدء: 2026-02-25

### ما تم إنجازه:

تم بناء تطبيق "مُراسِل" بالكامل - لوحة تحكم ذكية لتجميع وتنظيم أخبار التقنية باللغة العربية باستخدام الذكاء الاصطناعي.

#### الملفات المنشأة:

1. **database.py** - قاعدة بيانات SQLite مع الجداول:
   - `news` - تخزين الأخبار (العنوان، الملخص، المقال، المصدر، الحالة)
   - `sources` - المصادر (33 مصدر RSS افتراضي)
   - `settings` - الإعدادات

2. **config.py** - تحميل الإعدادات من ملف .env

3. **fetcher.py** - منطق جلب RSS وتلخيص الأخبار:
   - جلب من 33 مصدر RSS
   - تلخيص وترجمة بالذكاء الاصطناعي
   - تصنيف تلقائي للأخبار

4. **ai_writer.py** - كتابة المقالات الإخبارية:
   - مقالات 150-250 كلمة
   - أسلوب صحفي احترافي
   - دعم GPT-4o و Ollama

5. **main.py** - تطبيق FastAPI مع API endpoints:
   - GET/POST /api/news
   - POST /api/news/{id}/approve
   - POST /api/news/{id}/reject
   - /api/sources
   - /api/settings

6. **scheduler.py** - جدولة جلب الأخبار تلقائياً

7. **static/index.html** - واجهة المستخدم RTL بالعربية

8. **static/app.js** - منطق الواجهة الأمامية

9. **requirements.txt** - المكتبات المطلوبة

10. **.env** - قالب الإعدادات

11. **README.md** - توثيق بالعربية

### لتشغيل التطبيق:

```bash
cd murrasil
pip install -r requirements.txt
# أضف مفتاح OpenAI API في ملف .env
python main.py
```

ثم افتح المتصفح على: http://127.0.0.1:8000
